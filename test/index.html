Benefits of New Approach

  • WordPress Friendly: The <ins> tag can be easily inserted into content areas
  • Automatic Styling: Script handles all styling - no need for inline styles
  • Mobile-like Display: Proper aspect ratio and mobile-like appearance
  • Decoupled: Script is loaded once in head, embeds are defined in body
  • Dynamic: Supports dynamically added content via MutationObserver
  • Clean: No need to pair script tags with div elements
  • Flexible: Easy to add multiple embeds without script tag duplication
